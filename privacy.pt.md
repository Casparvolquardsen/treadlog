# Política de privacidade do TreadLog

**Última atualização: 20 de março de 2026**

## Visão geral

TreadLog é um aplicativo para iPhone para acompanhar a quilometragem dos seus tênis de corrida. Esta política de privacidade explica quais dados o aplicativo acessa, como são utilizados e quais são os seus direitos como utilizador.

## Dados que acedemos

### Dados do HealthKit

O TreadLog solicita acesso apenas de leitura aos seguintes dados do HealthKit:

- **Treinos** — tipo, distância, duração e data dos treinos de corrida
- **Dados de rota** — coordenadas GPS associadas aos treinos, usadas para exibir mapas de rotas

Este acesso é usado exclusivamente para exibir o seu histórico de treinos e calcular a quilometragem acumulada em cada par de tênis.

## Como os seus dados são utilizados

- Todo o processamento de dados ocorre **inteiramente no seu dispositivo**.
- Os dados do HealthKit **nunca são transmitidos para fora do seu dispositivo**.
- Os dados do HealthKit **nunca são partilhados com terceiros**.
- Os dados do HealthKit **nunca são utilizados para publicidade ou análises**.

## Armazenamento de dados

Os dados de tênis e atribuições de treinos que introduz no TreadLog são armazenados localmente no seu dispositivo usando o framework SwiftData da Apple. Esses dados nunca saem do seu dispositivo.

## Eliminação de dados

Pode eliminar todos os dados do aplicativo removendo o TreadLog do seu dispositivo. Pode revogar o acesso ao HealthKit a qualquer momento nas Definições do iOS em **Privacidade e segurança > Saúde > TreadLog**.

## Privacidade das crianças

O TreadLog não se destina a crianças menores de 13 anos e não recolhe conscientemente dados de crianças.

## Alterações a esta política

Se esta política sofrer alterações significativas, a versão atualizada será publicada neste URL com uma data revista.
